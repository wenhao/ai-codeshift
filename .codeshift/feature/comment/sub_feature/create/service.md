# 商品评价_创建(service)
## service粒度
- 独立的商品评价_创建service类，ProductCommentCreateService。

## 方法描述
- 创建商品评价

## 业务逻辑
1. 传入`创建商品评价-请求参数(Body)`对象。
2. 参数校验：根据传入的`创建商品评价-请求参数(Body)-商品SKU编号`使用`商品SKU_查询(mapper)`的服务对象查询数据库校验`商品SKU编号`是否真实存在，若不存在则抛出`商品 SKU 不存在`业务异常，错误码：`1_008_006_000`；
3. 参数校验：根据校验通过的`商品SKU编号`使用`商品SPU_查询(mapper)`的服务对象查询数据库校验`商品SPU编号`是否真实存在，若不存在则抛出`商品 SPU 不存在`业务异常，错误码：`1_008_005_000`；
4. 数据转换：将传入的`创建商品评价-请求参数(Body)`、校验通过的`商品SPU(entity)`、`商品SKU(entity)`，转换为数据库对应的`商品评价(entity)`数据对象。
5. 数据库操作：使用`商品评价_创建(mapper)`的数据库操作对象将`商品评价(entity)`数据对象插入数据库。
6. 方法返回：插入成功返回true。
