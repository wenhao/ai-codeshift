# 商品评价_创建(API)
## Controller粒度
- 独立的商品评价_创建Controller类，ProductCommentCreateController

### 接口描述
创建商品评价

### 请求地址
POST /product/comment/create

### 请求方法
POST

### 请求头
| 参数名        | 类型   | 是否必填 | 说明               |
|--------------|--------|----------|--------------------|
| Content-Type | String | 是       | 固定值：application/json |

### 请求参数(Body)
| 参数名            | 类型         | 是否必填 | 说明                                   | 校验说明                                                                 |
|------------------|--------------|----------|----------------------------------------|--------------------------------------------------------------------------|
| userId           | Long         | 是       | 评价人编号                             | 仅非空校验，校验不通过提示：评价人编号不能为空                             |
| orderItemId      | Long         | 是       | 评价订单项编号                         | 仅非空校验，校验不通过提示：评价订单项编号不能为空                         |
| userNickname     | String       | 是       | 评价人名称                             | 仅非空校验，校验不通过提示：评价人名称不能为空                             |
| userAvatar       | String       | 是       | 评价人头像                             | 仅非空校验，校验不通过提示：评价人头像不能为空                             |
| skuId            | Long         | 是       | 商品SKU编号                            | 仅非空校验，校验不通过提示：商品SKU编号不能为空                             |
| descriptionScores| Integer      | 是       | 描述星级 1-5分                         | 仅非空校验，校验不通过提示：描述星级不能为空                               |
| benefitScores    | Integer      | 是       | 服务星级 1-5分                         | 仅非空校验，校验不通过提示：服务星级分不能为空                             |
| content          | String       | 是       | 评论内容                               | 仅非空校验，校验不通过提示：评论内容不能为空                               |
| picUrls          | List<String> | 否       | 评论图片地址数组，最多上传9张          | 长度校验，最大长度9，校验不通过提示：评论图片地址数组长度不能超过9张       |

### 响应参数
| 参数名  | 类型    | 说明               |
|--------|---------|--------------------|
| code   | int     | 响应码，200=成功    |
| msg    | String  | 响应信息           |
| data   | Boolean | 创建成功返回true    |
```
