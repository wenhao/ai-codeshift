# 商品评价_分页查询(service)

## service粒度
- 独立的商品评价_分页查询Service 类：`ProductCommentPageService`。

## 方法描述
分页查询商品评价列表，支持多条件组合筛选，返回标准分页结果。

## 业务逻辑
1. 数据库操作：根据`商品评价_分页查询(API)-请求参数（Query）`所有参数使用`商品评价_分页查询(Mapper)`的数据库操作对象查询所有`商品评价(entity)`列表，可按以下参数组合查询数据。
   * 可按分页参数查询数据，`页码`默认为1和`每页条数`默认为10查询；
   * 若查询参数`用户昵称`不为空，则模糊查询；
   * 若查询参数`订单编号`不为空，则精确查询；
   * 若查询参数`商品SPU编号`不为空，则精确查询；
   * 若查询参数`评分`不为空，则精确查询；
   * 若查询参数`回复状态`不为空，则精确查询；
   * 若查询参数`商品SPU名称`不为空，则模糊查询；
   * 若查询参数`创建时间`不为空，则按创建时间范围查询；
   * 按`评价编号`倒序排序；
2. 数据转换：将查询到的`商品评价(entity)`分页数据，转换为`商品评价_分页查询(api)-响应参数`数据对象。
3. 方法返回：返回`商品评价_分页查询(api)-响应参数`数据对象。
