# 商品评价_回复(service)

## service粒度
- 独立的商品评论_回复Service类，ProductCommentReplyService

## 方法描述
- 商家对商品评价进行回复

## 业务逻辑
1. 传入`商品评价_回复(API)-请求参数(Body)`对象；
2. 数据校验：根据请求参数中的商品`评论编号`，调用`商品评价_回复(mapper)`的数据库操作对象查询数据库校验`评论编号`是否真实存在，若不存在抛出`商品评价不存在`业务异常，错误码：`1_008_007_000`；
3. 数据库操作：根据上个步骤获取的`商品评价(entity)`，通过`商品评价_回复(API)-请求参数(Body)`设置`商家回复内容`、`商家回复时间`为当前时间、`回复管理员编号`、`商家是否回复`为true，调用`商品评价_回复(mapper)`更新`商品评价(entity)`；
4. 方法执行：更新成功返回true。
