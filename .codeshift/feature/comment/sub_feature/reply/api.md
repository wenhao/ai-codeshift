# 商品评价_回复(api)

## controller粒度
- 独立的商品评论_回复Controller类，ProductCommentReplyController。

### 接口描述
商家回复商品评价

### 请求地址
PUT /product/comment/reply

### 请求方法
PUT

### 请求头
| 参数名        | 类型   | 是否必填 | 说明               |
|--------------|--------|----------|--------------------|
| Content-Type | String | 是       | 固定值：application/json |

### 请求参数(Body)
| 参数名          | 类型    | 是否必填 | 说明     | 校验说明 |
|--------------|---------|----------|--------|----------|
| id           | Long    | 是       | 评论编号   | 非空校验，校验不通过提示：评论编号不能为空 |
| replyUserId       | Long    | 是       | 回复管理员编号   | 非空校验，校验不通过提示：回复管理员编号不能为空 |
| replyContent | String  | 是       | 商家回复内容 | 非空校验，校验不通过提示：商家回复内容不能为空 |

### 响应参数
| 参数名  | 类型    | 说明               |
|--------|---------|--------------------|
| code   | int     | 响应码，200=成功    |
| msg    | String  | 响应信息           |
| data   | Boolean | 回复是否成功       |
