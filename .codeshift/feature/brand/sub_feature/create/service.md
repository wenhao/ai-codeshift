# 商品品牌_创建(service)
## service粒度
* 独立的商品品牌_创建service类，ProductBrandCreateService。

## 方法描述
* 创建商品品牌

## 业务逻辑
1. 传入`创建商品品牌-请求参数(Body)`对象。
2. 参数校验：根据传入的`创建商品品牌-请求参数(Body)-品牌名称`使用`商品品牌(mapper)`的数据库操作对象校验品牌名称是否已存在，若已存在则抛出“品牌名称已存在”业务异常，错误码：1_008_002_002。
3. 数据转换：将传入的`创建商品品牌-请求参数(Body)`对象，转换为数据库对应的`商品品牌(entity)`数据对象。
4. 数据库操作：使用`商品品牌(mapper)`的数据库操作对象将`商品品牌(entity)`数据对象插入数据库。
5. 方法返回：返回数据库自动生成的商品`商品品牌(entity)-品牌编号`。
